## Description

This project takes text input from the user and reads it aloud using Python.  
It uses **pyttsx3**, a cross-platform text-to-speech library that works offline.

---

## Features

- Convert any text input into speech
- Works offline (no internet needed)
- Easy command-line interface
- Exit program by typing `exit`

---

## Requirements

- Python 3.x  
- `pyttsx3` library
